Pathfinding: Accelerated A* Algorithm

I developed a pathfinding solution for the Unreal Engine, based on the A* Algorithm for 3D environments, utilizing a 2D grid map to store nodes. These nodes are either pre-collected from meshes at the start of the game or dynamically added asynchronously during gameplay. This ensures that a route, such as one guiding a bot to the player, is always successfully determined. With tangential checks the number of possible routes is minimized. The Graph can be stored and loaded from storage as binary file.

Terrain generation: Bezier splines

Im developing on a Terrain generation solution for the Unreal Engine, using connected Bezier splines, including generation of simple trees and their meshes. To keep performance in mind, I merge similar meshes, like all leafs into a single Buffer per Chunk. Gpu Instancing, to spawn grass on the terrain, is also supported.

Special Features: Dynamic mesh deformation on all procedural layers, Custom Lod System, saving and loading terrain by writing custom binaries, and Meshdata conversion to a UStaticMesh assets.
