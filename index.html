Pathfinding: A* Algorithm

I developed a pathfinding solution for the Unreal Engine, based on the A* Algorithm for 3D environments, utilizing a 2D grid map to store nodes. These nodes are either pre-collected from meshes at the start of the game or dynamically added asynchronously during gameplay. This ensures that a route, such as one guiding a bot to the player, is always successfully determined.
